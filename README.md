# C Programming Mathematical Programs

A collection of basic mathematical programs written in C. This repository is designed for beginners to practice problem-solving, mathematical concepts, and fundamental C programming skills.

## Programs Included

- Cube of a Number (`cube.c`)
- Function Prototype Example (`photo-types.c`)
- Combination Calculator (`combinations.c`)
- Pascal Triangle (`pascal--triangles.c`)
- Swap Two Numbers (`swap.c`)

## Features

- Beginner-friendly C programs
- Simple and well-structured code
- Easy to understand logic
- Practice for loops, functions, and mathematical concepts

## Technologies Used

- C Programming Language
- GCC Compiler / Visual Studio Code

## Learning Topics

- Functions
- Function Prototypes
- Mathematical Calculations
- Pascal Triangle
- Number Manipulation
- Basic Problem Solving

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Ayan-nawaz/c-programming-mathematical-programs.git
   ```

2. Open the project in Visual Studio Code or any C IDE.

3. Compile the program:
   ```bash
   gcc filename.c -o output
   ```

4. Run the executable:
   ```bash
   ./output
   ```
